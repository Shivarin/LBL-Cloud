| Файл | Листинг | Оригинал на сервере |
|------|---------|---------------------|
| `main_auth_2fa_excerpt.py` | А.13 — 2FA, смена пароля | `backend/main.py` |
| `main_login_2fa_excerpt.py` | Вход с 2FA (текст §3.3) | `backend/main.py` |
| `utils_email_cloud_excerpt.py` | А.15 — бренд Cloud в письмах | `backend/utils.py` |

Полные модули Cloud в репозитории: `../lbl_drive.py`, `../cloud_billing.py`.
